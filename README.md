# jtracker
job applications tracker

```

  $ pip install -r requirements.txt
  $ ./manage.py makemigrations
  $ ./manage.py migrate
  $ ./manage.py createsuperuser  #follow instructions
  $ ./manage.py runserver
  $ # visit http://localhost:8000 in browser, login with created credentials

```
